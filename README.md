# 🏋️‍♂️ Sweat Factory AI Agent - Atención al Cliente y Ventas

Este repositorio contiene la configuración, base de conocimientos y directrices del agente de Inteligencia Artificial diseñado para el gimnasio **Sweat Factory**.

## 🎯 Objetivo del Proyecto
Automatizar la atención al cliente en canales digitales, resolviendo dudas frecuentes sobre el reglamento, horarios y servicios, además de actuar como un canal activo de ventas y prospección.

## 🛠️ Stack Tecnológico
* **Plataforma de Conversación:** [Coloca aquí la plataforma, ej. Voiceflow / ManyChat]
* **Motor de IA:** OpenAI GPT-4o (a través de API)
* **Base de Conocimientos:** Documentación oficial en PDF de Sweat Factory

## 📁 Estructura del Repositorio
* `/agent-config`: Contiene el archivo JSON exportado con la lógica y bloques del bot.
* `/knowledge-base`: Documentos de texto y PDF que alimentan el cerebro de la IA (Reglamento, Precios, FAQs).

## 🚀 Cómo Replicar este Agente
1. Descarga el archivo `.json` de la carpeta `/agent-config`.
2. Importa el archivo en tu cuenta de [Voiceflow/ManyChat/Botpress].
3. Sube el PDF de `/knowledge-base` a la sección de Base de Conocimientos de la plataforma.
4. Vincula tu API Key de OpenAI y ¡listo!
